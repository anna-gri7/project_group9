# Меблерія 🛋️

## О проєкті

Сучасний інтернет-магазин меблів з повною інтеграцією API. Адаптивна верстка,
модальні вікна, слайдери, фільтрація та форми замовлення.

**Задача**: Створити landing page меблевого магазину з інтеграцією бекенду.

## Технології

```
HTML5 | CSS3 | JavaScript ES6+
Vite | Axios | Swiper.js | iziToast | rater-js | accordion-js
Адаптивність: 375px+ | 768px | 1440px
```

## API

```
https://furniture-store-v2.b.goit.study/api-docs/
• /categories - категорії
• /furnitures - товари + пагінація
• /furniture/{id} - деталі
• /feedbacks - відгуки
• POST /orders - замовлення
```

## Запуск проєкту

```bash
npm install
npm run dev
```

**Live Server**: http://localhost:5173
